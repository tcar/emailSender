# emailSender
Application that sends random Chuck Noris jokes to selected emails

## Installation

For building and using emailSender node 10.15.1 is required:

    # install latest npm
    npm install -g npm

    # install n package manager
    npm install -g n

    # install latest node
    n latest # or 10.15.1

Install emailSender

    npm install
    
Start emailSender

    # Start backend
    node index.js

    # Run client
    npm run webpack

    # Run tests
    npm run test-all
